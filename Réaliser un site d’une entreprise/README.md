## Application pour la gestion du capital humain des entreprises

## Contexte du projet

```bash
Le brief est constitué de 2 parties :

1ére page contient :

les éléments standards d'une application web(menu, inscription,login,logout...)
Une liste des entreprises.
Chaque entreprise contient des départements.
les actions possibles :
Ajout d'une nouvelle entreprise et ces départements.
Ajout d'un nouveau département pour une entreprise existante.
Redirection vers la 2eme page suite à un clic sur un département.
2éme page contient :

les informations du département choisi depuis la 1ére page.
le département contient des salariés.
Chaque salarié est identifié par un matricule.
les actions possibles :
Ajout d'un nouveau salarié et ces informations.
3éme page :

qui permet de chercher un salarié par son nom.
Caractéristiques des éléments :

Entreprise : Nom, local, description, liste des départements,...
Département : Nom, chef département, description, liste des matricules(les salariés)...
Salarié : Matricule, Nom, Prenom, Age, salaire, ...
Les technologie à utiliser :

Front end : HTML/CSS/Javascript/Jquery/Ajax/Bootstrap.
Back end : Node JS, JSON(à vous de définir la structure), JSON pour les comptes après inscription.
Vous êtes amenés à :

Être créatif en proposant un contenu adapté (texte correcte /et images)
respecter UI/UX et et l’ergonomie de votre site.
travailler sur le responsive design.
SEO de votre site .
héberger votre site sur GitHub.
Livrables attendus
Tableau trello décrivant la répartition et l’évolution des tâches afin atteindre l’objectif.



```
