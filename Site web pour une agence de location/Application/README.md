## Site-web-pour-une-agence-de-location

## Contexte du projet

```bash

Votre client est une entreprise de location de (voiture ,maison,,motocycle pour les amateurs:) ,....)vous avez le libre choix.

votre site doit être doté de :

une page vitrine qui comportent une galerie de votre produit (voiture,motocycle,maison,....) avec animation slide javascript.
une page d’inscription interactive qui demande le nom et le password et Confirmation de password (itération tant qu'il ne sont pas identique). vous pouvez ajouter d’autre option.
page réservation du produit (image produit, nom client, date naissance client,période de location,nbr personne ...).
Après validation une zone s’affiche en guise de compte-rendu/confirmation des données saisie.(image produit, nom client, age client ,période de location,nbr personne .. )

N.B:

Tous les champs saisie doivent être en Majuscule.

vous êtes amenés à utiliser l’HTML,le préprocesseur du CSS SASS, le javascript et bien sur Trello pour gestion de votre projet.

Respecter le responsive design


```
